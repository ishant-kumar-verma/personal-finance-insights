## 📌 Overview

Personal Finance Insights is a modern, interactive Power BI dashboard designed to help users understand their financial behavior through intuitive visualizations and dynamic analytics.

The dashboard transforms raw transaction data into meaningful insights, enabling users to monitor income, expenses, savings, and spending patterns while receiving intelligent financial recommendations.

---

## 🎯 Business Problem

Managing personal finances can be challenging when transaction data is scattered and difficult to interpret.

This dashboard addresses that challenge by providing a centralized view of:

- Income vs Expenses
- Cash Flow Trends
- Expense Distribution
- Spending Categories
- Savings Performance
- Financial Intelligence

allowing users to make better financial decisions.

## 📊 Dashboard Features

### 📊 Expense Analysis

### 🧠 Financial Intelligence

Dynamic insights generated using DAX logic.

Examples include:

- Financial Health Status
- Highest Expense Category
- Savings Rate Analysis
- Personalized Financial Recommendation

---

## 📂 Dataset

The dashboard is built using a personal finance transaction dataset containing:

- Date
- Transaction Description
- Category
- Amount
- Transaction Type (Income / Expense)

Time Period:

**January 2020 – December 2024**
